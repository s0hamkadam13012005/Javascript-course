Generate Computer's Move Inside the playGame function, the pickComputerMove function is called: javascript Copy code const computerMove = pickComputerMove(); pickComputerMove: Generates a random number. 3. Decide the Winner The playGame function compares: Player's move (e.g., 'rock') and Computer's move (e.g., 'scissors'). Based on the rules of Rock-Paper-Scissors, it sets the result ('You win.', 'You lose.', or 'Tie.').
4. Display the Result The result, along with both moves, is shown using an alert: javascript Copy code alert(`You picked ${playerMove}.
